# Compatibility Lab

This folder contains small sample files for checking Cagnard file type detection and in-app openers.

Suggested checks:

- `developer-guide.md`: Markdown rendered/source views.
- `payload.json`: JSON tree/source views plus prettify and minify.
- `inventory.csv`: CSV table view plus raw view.
- `metrics.tsv`: TSV table view.
- `settings.yaml` and `sample.xml`: structured text/source handling.
- `ServerExample.scala`: source code handling.
- `server.log`: log/source handling.
- `diagram.svg`: browser-native image handling.
- `unknown.payload`: unsupported-file fallback.
